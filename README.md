# My Hemlock project

This project was created with [Hemlock](https://dsbowen.gitlab.io/hemlock) - a python framework for creating surveys and behavioral experiments.

## Run the app locally

```bash
$ python app.py
```

Or

```bash
$ gunicorn --worker-class eventlet --workers 1 app:app
```
